# Signup Calendar Block

A WordPress Gutenberg block plugin for displaying signup calendars.

## Installation

1. Install dependencies:
```bash
npm install
```

2. Build the block:
```bash
npm run build
```

For development with auto-rebuild:
```bash
npm start
```

## Usage

1. Activate the plugin in WordPress
2. Add the "Signup Calendar" block to any post or page
3. The block will display "Hello" when inserted

## Development

- `npm start` - Start development mode with auto-rebuild
- `npm run build` - Build for production
- `npm run format` - Format code
- `npm run lint:js` - Lint JavaScript
- `npm run lint:css` - Lint CSS
